## Run Galaxy Locally üöÄ

##### Install Galaxy
```bash
git clone https://github.com/r480github/GalaxyV4.git
cd GalaxyV4
```


<br>

##### Install dependencies

```bash
npm i  
```

##### Start

```bash
npm start
```
#### NOTE: The dafault port for GalaxyV4 is 3000. You can visit it by typing localhost:3000/ into a web browser
